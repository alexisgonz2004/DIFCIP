# Dominican Integrated Fiscal and Commercial Intelligence Platform (DIFCIP)

## 📋 Descripción

Plataforma analítica que procesa datos de ventas de una empresa dominicana, generando análisis de rentabilidad, segmentación de clientes y validaciones fiscales (ITBIS, NCF).

## 🛠️ Tecnologías utilizadas

- PostgreSQL
- Python (Pandas)
- Microsoft Excel
- Power BI
- Git / GitHub

└── README.md

## 🎯 Objetivos del proyecto

- Análisis de ventas y rentabilidad
- Segmentación de clientes
- Validaciones fiscales (ITBIS, NCF)
- Dashboard ejecutivo en Power BI
- Automatización con Python
